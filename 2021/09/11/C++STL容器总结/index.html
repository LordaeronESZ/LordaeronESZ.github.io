C++STL容器总结

发表于2021-09-11|更新于2021-09-16|C++
|字数总计:1.3k|阅读时长:5分钟|阅读量:

本文将对常用的C++标准库中的容器类做一个总结，便于学习记忆以及今后的查阅。

string

at

char &at(size_t _Off)

与下标访问类似，返回指定位字符的引用。不同之处为对于下标访问，字符串的末尾元素有效且返回’\0’，且如果越界访问则会未定义行为，而使用at进行访问不包含末尾元素，且越界访问会抛出out_of_range异常。

string s("abc");
cout << s[3] << endl; //输出'\0'
cout << s.at(3) << endl; //抛出out_of_range异常

front、back

char &front()
char &back()

返回头字符和尾字符的引用。

string s("abc");
cout << s.front() << endl; //输出'a'
cout << s.back() << endl; //输出'c'

begin、end

string::iterator begin()
string::iterator end()

返回头字符和尾字符的迭代器类型，可用以对字符串进行遍历和修改。迭代器(iterator)是类模板，对指针进行了封装和抽象，在使用上与指针类似，用“*”访问所指向的元素。

string s("abc");
for(auto it = s.begin(); it < s.end(); it++)
    cout << *it;	//遍历并输出s各字符

length、size

size_t length() const
size_t size() const

length和size都是返回字符串的长度，二者没有区别，length是沿用了C语言的习惯而被保留下来的。

string s("abc");
cout << s.size() << "," << s.length(); //输出3,3

clear

void clear()

清空字符串，无需赘述。

empty

bool empty() const

判断字符串是否为空，为空返回true，非空返回false。

erase

//删除指定迭代器位置_Where的字符，返回下个元素的迭代器
string::iterator erase(string::const_interator _Where)

//删除迭代器_First和_Last之间的字符（不包含_Last），返回_Last位置的迭代器
string::iterator erase(string::const_interator _First, string::const_interator _Last)    

//删除从下标_Off开始后的_Count个字符，返回修改后字符串的引用
string &erase(size_t _Off, size_t _Count)    

//删除下标_Off之后的所有字符，返回修改后字符串的引用
string &erase(size_t _Off = 0U)

示例如下： 

string s = "abc";
cout << *s.erase(s.begin()) << endl; //输出b
s = "abc";
cout << *s.erase(s.begin(), s.begin() + 2) << endl; //输出c
s = "abc";
cout << s.erase(1, 1) << endl; //输出ac
s = "abc";
cout << s.erase(1) << endl; //输出a

find

//查找字符串从下标_Off开始，第一个_Ch字符的下标位置，并将其返回
size_t find(char _Ch, size_t _Off = 0U) const
    
//查找字符串从下标_Off开始，第一个子串_Right的下标开始位置，并将其返回    
size_t find(string &_Right, size_t _Off = 0U) const

find函数若是没有找到相匹配的字符或字符串，则会返回一个常数string::npos，可用if (str.find(_Ch, _Off) != string::npos)来判断是否查找到目标值。示例如下：

string s("abc");
cout << s.find('c', 1) << endl; //输出2
cout << s.find("bc", 0) << endl; //输出1
cout << s.find('d', 0) << endl; //输出string::npos

insert

//在下标_Off的位置，插入字符串_Right，并返回修改后的字符串的引用
string &insert(size_t _Off, string &_Right)

//在下标_Off的位置，插入_Count个字符_Ch，并返回修改后字符串的引用
string &insert(size_t _Off, size_t _Count, char _Ch)

//在迭代器_Where的位置，插入_Count个字符_Ch，并返回插入位置的迭代器
string::iterator insert(string::const_iterator _Where, size_t _Count, char _Ch)   

//在迭代器_Where的位置，插入单个字符_Ch，并返回插入位置的迭代器
string::iterator insert(string::const_iterator _Where, char _Ch)

示例如下：

string s("abc");
cout << s.insert(1, "dd") << endl; //输出addbc
cout << s.insert(3, 'e') << endl; //输出addebc
cout << *s.insert(s.begin(), 1, 'f') << endl; //输出f
auto it = s.insert(s.begin() + 1, 'g');
for(; it < s.end(); it++) //输出gddebc
    cout << *it;

replace

//将迭代器_First和_Last之间的字符串子串替换成_Count个字符_Ch，并返回修改后字符串的引用
string &replace(string::const_iterator _First, string::const_iterator _Last, size_t _Count, char _Ch)

//将迭代器_First和_Last之间的字符串子串替换成字符串_Right，并返回修改后字符串的引用
string &replace(string::const_iterator _First, string::const_iterator _Last, string &_Right)

//将从下标_Off开始长度为_Nx的子串替换成_Count个字符_Ch，并返回修改后字符串的引用    
string &replace(size_t _Off, size_t _Nx, size_t _Count, char _Ch)

//将从下标_Off开始长度为_Nx的子串替换成字符串_Right，并返回修改后字符串的引用  
string &replace(size_t _Off, size_t _Nx, string &_Right)

示例如下：

string s("abc");
cout << s.replace(s.begin(), s.end(), 3, 'd'); //输出ddd
cout << s.replace(s.begin(), s.end(), "cba"); //输出cba
cout << s.replace(0, 1, 2, 'd'); //输出ddba
cout << s.replace(2, 2, "ee"); //输出ddee

pop_back、push_back

//删除字符串尾元素
void pop_back()

//向字符串尾添加字符_Ch
void push_back(char _Ch)

substr

string sub_str(size_t _Off = 0U, size_t _Count = 4294967295U)

从下标_Off开始，截取长度为_Count的字符串，并返回。示例如下：

string s("abc");
string sub_s = s.substr(0, 2);
cout << sub_s << endl; //输出字符串ab
