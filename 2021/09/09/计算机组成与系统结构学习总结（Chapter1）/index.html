第一章 计算机系统概论

计算机的分类

模拟计算机

“模拟”就是相似的意思。模拟计算机的特点是数值由连续量来表示，运算过程也是连续的。

数字计算机

它是在算盘的基础上发展起来的，是用数目字来表示数量的大小。数字计算机的主要特点是按位运算，并且不连续地跳动计算。

主要区别

数字计算机 模拟计算机
数据表示方式 数字0/1 电压
计算方式 数字计数 电压组合和测量值
控制方式 程序控制 盘上连线
精度 高 低
数据存储量 大 小
逻辑判断能力 强 无

电子模拟计算机由于精度与解题能力都有限，因此应用范围较小。电子计算机则不同，它是以近似于人类的“思维过程”来进行工作的，因此被称为电脑。

计算机的发展简史

计算机的五代变化

  1. 1946-1957年，电子管计算机：数据处理
  2. 1958-1964年，晶体管计算机：工业控制
  3. 1965-1971年，中小规模集成电路计算机：小型计算机
  4. 1972-1990年，大规模和超大规模集成电路计算机：微型计算机
  5. 1991年~，巨大规模集成电路计算机：单片机

摩尔定律

1965年英特尔创始人之一的戈登·摩尔观察到芯片上的晶体管数量每年翻一番，1970年这种态势减慢成每18个月翻一番，这就是人们所称的摩尔定律。值得一提的是，摩尔定律并不是物理定律，而更像是对芯片行业发展的预测或者说是对行业的一种要求。

半导体存储器的发展

  1. 20世纪50-60年代，所有计算机的存储器都是由微小的铁磁体环组成。
  2. 1970年，仙童半导体公司生产出了第一个较大容量半导体存储器。一个相当于单个磁芯大小的芯片，包含了256位的存储器。
  3. 从1970年起，半导体存储器经历了11代：单个芯片1KB、4KB、16KB、64KB、256KB、1MB、4MB、16MB、64MB、256MB、1GB，每一代相比前一代存储密度提高4倍，而每位价格和存取时间都在下降。

微处理器的发展

  1. 1971年，Intel公司开发出Intel 4004。这是第一个将CPU的所有元件都放入同一块芯片内的产品

  2. 1972年，Intel 8008，这是第一个8位微处理器，它比4004复杂一倍

  3. 1974年，Intel 8080。这是第一个通用微处理器

  4. 20世纪70年代末，通用16位微处理器，Intel 8086是其中之一

  5. 1981年，贝尔实验室和HP公司开发出了32位单片微处理器

  6. 1985年，推出了32位微处理器Intel 80386

  7. 现在的64位处理器和多核处理器

计算机的性能指标

  • 吞吐量：表征一台计算机在某一时间间隔内能够处理的信息量，单位是字节/秒（B/S）

  • 响应时间：表征从输入有效到系统产生响应之间的时间度量，用时间单位来度量

  • 利用率：表示在给定的时间间隔内，系统被实际使用的时间所占的比率，一般用百分比表示

  • 处理机字长：指处理机运算器中一次能够完成二进制数运算的位数。当前处理机的字长有8位、16位、32位、64位字长越长，表示计算的精度越高

  • 总线宽度：一般指CPU中运算器与存储器之间进行互连的内部总线二进制位数

  • 存储器容量：存储器中所有存储单元的总数目，通常用KB、MB、GB、TB来表示

  • 存储器带宽：存储器的速度指标，单位时间内从存储器读出的二进制数信息量，一般用字节数/秒表示

  • 主频/时钟周期：CPU的工作节拍受主时钟控制，主时钟不断产生固定频率的时钟，主时钟的频率（f）叫CPU的主频。度量单位是MHz、GHz。主频的倒数称为CPU时钟周期，T = 1 / f.

  • CPU执行时间：表示CPU执行一般程序所占用的CPU时间，计算公式如下：
    CPU执行时间 = CPU时钟周期数 * CPU时钟周期

  • CPI：表示每条指令周期数，即执行一条指令所需的平均时钟周期数，计算公式如下：
    CPI = 执行某段程序所需的CPU时钟周期数 / 该程序包含的指令条数

  • MIPS：表示每秒执行多少百万条定点指令数，计算公式如下：
    MIPS = 指令数 / (程序执行时间 * 10^6)

  • FLOPS：表示每秒执行浮点操作的次数，用来衡量机器浮点操作的性能，计算公式如下：
    FLOPS = 程序中的浮点操作次数 / 程序执行时间

计算机的硬件

硬件组成要素

计算机硬件系统的五大功能部件有：

  • 控制器：人的大脑的操作控制功能
  • 运算器：人的大脑的计算功能
  • 存储器：人的大脑的记忆功能
  • 输入设备：交互接口，笔
  • 输出设备：交互接口，纸

运算器

运算器就像是一个由电子线路构成的算盘，它的主要功能是进行加、减、乘、除等算术运算。除此之外，还可以进行逻辑运算，因此通常被称为ALU（算数逻辑运算部件）。

人类习惯于十进制运算，但考虑到电子器件的特性，计算机通常采用二进制数。0 和 1 可以用电压的高低、脉冲的有无来表示，因此实现较为简单。

运算器由：加法器、移位门、寄存器组、输入选择门和数据总线组成。

存储器

存储器的功能是保存或“记忆”解题的原始数据和解题步骤。为此，需要将参加运算的数据和解题步骤通过输入设备送到存储器中保存起来。

由于计算机中保存的数据都是二进制形式，目前采用半导体存储器来进行存储，如果一个数用 16 位二进制代码来表示，就需要有16个触发器来保存这些代码。通常在存储器中把保存这个数的16个触发器称为一个存储单元。存储器有许多的存储单元组成，每个存储单元都有编号，称之为地址。在需要获取某个数据时，就需要根据该数据的地址查找相应的存储单元。

半导体存储器的存储容量有限，因此计算机中配备了存储容量更大的磁盘存储器和光盘存储器，称之为外存储器，相对的，半导体存储器称之为内存储器，简称内存。

控制器

控制器是计算机中发号施令的部件，它控制计算机的各部件有条不紊地进行工作。更具体地讲，控制器的任务是从内存中取出解题步骤加以分析，然后执行某种操作。

计算程序

运算器只能完成简单的加减乘除等其他一些简单操作。对于一个复杂的问题，计算机需要在交由运算器运算前将其拆分为若干个加减乘除的基本操作来做。每一个基本操作称为一条指令，而求解某一某一问题的一串指令序列，称之为该问题的计算程序，简称程序。

指令的形式

每条指令应当明确告诉控制器，从存储器的哪个单元取数，并进行何种操作。由此可知指令由两部分组成，即操作的性质和操作数的地址。前者称为操作码，后者称为地址码。

指令数码化后，就可以和数据一样放入存储器。像这样，如果指令和数据放在同一个存储器中，称为冯·诺依曼结构，如果指令和数据分别存放在两个存储器，称为哈佛结构。显然后者结构的计算机运算速度更快。

输入输出设备

  • 输入设备把程序和数据转化为计算机能识别和处理的数据形式的设备，输入的程序和数据存入计算机存储器中。
  • 输出设备是将计算机中的二进制信算转化为用户所需要的数据格式的设备。它将计算机中的信息（计算结果等）以十进制或字符、图形、表格等形式显示或打印出来。

除了上述各部件外，计算机系统还必须有总线。系统总线是构成计算机系统的骨架，是多个系统部件之间进行数据传送的公共通路。借助系统总线，计算机在各系统部件之间实现地址、数据和控制信息的操作。

计算机的软件

软件的组成和分类

计算机软件一般分为两大类，一类叫系统程序，一类叫应用程序。

  • 系统程序用来简化程序设计，简化使用方法，提高计算机的使用效率，发挥和扩大计算机的功能和用途。主要包括以下四类：

    （1）各种服务性程序，如诊断程序、排错程序等；

    （2）语言程序，如汇编程序、编译程序、解释程序等；

    （3）操作系统；

    （4）数据库管理系统。

  • 应用程序是用户利用计算机来解决某些问题而编制的程序，如工程设计、数据处理程序等。

软件的发展演变

  • 编程语言的发展

（1）机器程序

（2）汇编语言

（3）算法语言

  • 系统软件的发展

（1）操作系统

（2）分布式系统软件

计算机系统的层次结构

  1. 1级 微程序设计级：直接由硬件执行
  2. 2级 一般机器级：微程序
  3. 3级 操作系统级：操作系统
  4. 4级 汇编语言级：汇编程序
  5. 5级 高级语言级：编译程序
