hexoеҚҡе®ўзҡ„еӨҮд»ҪдёҺжҒўеӨҚ

еҸ‘иЎЁдәҺ2021-08-25|жӣҙж–°дәҺ2021-09-02|з»ҸйӘҢ
|еӯ—ж•°жҖ»и®Ў:939|йҳ…иҜ»ж—¶й•ҝ:3еҲҶй’ҹ|йҳ…иҜ»йҮҸ:

дёәдәҶйҳІжӯўиҜҜж“ҚдҪңжҲ–и®Ўз®—жңәеҙ©жәғеҜјиҮҙзҡ„еҚҡе®ўжәҗж–Үд»¶дёўеӨұпјҢжҲ–ж»Ўи¶іжҲ‘д»¬жӣҙжҚўз”өи„‘иҝӣиЎҢеҶҷдҪңзҡ„йңҖжұӮпјҢе®ҡжңҹиҝӣиЎҢеӨҮд»ҪдёҺжҒўеӨҚе°ұжҳҫеҫ—еҫҲйҮҚиҰҒдәҶгҖӮ

еӨҮд»Ҫ

йңҖеӨҮд»Ҫзҡ„ж–Үд»¶зӣ®еҪ•

жҲ‘д»¬еңЁиҝӣиЎҢеӨҮд»Ҫж—¶пјҢе№¶дёҚйңҖиҰҒе°Ҷж•ҙдёӘеҚҡе®ўзӣ®еҪ•е…ЁйғЁеӨҮд»ҪпјҢеҸӘеӨҮд»Ҫд»ҘдёӢеҮ дёӘз”ЁжҲ·иҮӘе®ҡд№үзҡ„еҚіеҸҜпјҡscaffoldsзӣ®еҪ•пјҢsourceзӣ®еҪ•пјҢthemesзӣ®еҪ•пјҢ.gitignoreж–Үд»¶пјҢ_config.ymlж–Үд»¶пјҢpackage.jsonж–Үд»¶пјҢе…¶д»–йғҪжҳҜйҖҡз”Ёзҡ„гҖӮ

йҰ–е…ҲжҲ‘д»¬е…ҲжқҘи§ӮеҜҹж•ҙдёӘеҚҡе®ўзӣ®еҪ•зҡ„з»“жһ„пјҡ

  • .deploy_git жң¬зӣ®еҪ•дјҡеңЁжҲ‘д»¬йғЁзҪІж—¶з”ҹжҲҗпјҢд№ҹе°ұжҳҜgithubдёҠдҝқеӯҳзҡ„ж–Үд»¶пјҢж— йңҖеӨҮд»ҪгҖӮ

  • node_modules жң¬зӣ®еҪ•еҢ…еҗ«дәҶhexoеҚҡе®ўдҫқиө–зҡ„жЁЎеқ—пјҢе®үиЈ…ж—¶иҮӘеҠЁз”ҹжҲҗпјҢж— йңҖеӨҮд»ҪгҖӮ

  • public дёҺ.deploy_gitзұ»дјјпјҢжҳҜзј–иҜ‘еҗҺз”ҹжҲҗзҡ„ж–Үд»¶йқҷжҖҒзҪ‘йЎөж–Үд»¶пјҢж— йңҖеӨҮд»ҪгҖӮ

  • scaffolds дҝқеӯҳдәҶз”ЁжҲ·ж–Үз« зҡ„жЁЎжқҝпјҢйңҖиҰҒеӨҮд»ҪгҖӮ

  • source дҝқеӯҳдәҶз”ЁжҲ·ж–Үз« зҡ„жәҗж–Үд»¶пјҢйңҖиҰҒеӨҮд»ҪгҖӮ

  • themes дҝқеӯҳдәҶз”ЁжҲ·дёӢиҪҪзҡ„дё»йўҳж–Үд»¶дёҺй…ҚзҪ®пјҢйңҖиҰҒеӨҮд»ҪгҖӮ

  • .gitignore дҝқеӯҳдәҶжҺЁйҖҒеҲ°githubж—¶еҝҪз•Ҙзҡ„ж–Үд»¶еҗҚпјҢйңҖиҰҒеӨҮд»ҪгҖӮ

  • _config.yml дҝқеӯҳдәҶз”ЁжҲ·й…ҚзҪ®дҝЎжҒҜпјҢйңҖиҰҒеӨҮд»ҪгҖӮ

  • db.json дҝқеӯҳдәҶзҪ‘йЎөзҡ„ж•°жҚ®ж–Үд»¶пјҢеңЁзј–иҜ‘з”ҹжҲҗйқҷжҖҒзҪ‘йЎөж—¶дјҡиҮӘеҠЁжӣҙж–°пјҢж— йңҖеӨҮд»ҪгҖӮ

  • package.json дҝқеӯҳдәҶдҫқиө–зҡ„жЁЎеқ—еҲ—иЎЁпјҢйңҖиҰҒеӨҮд»ҪгҖӮ

  • package-lock.json дҝқеӯҳдәҶдҫқиө–зҡ„жЁЎеқ—е®үиЈ…и®°еҪ•пјҢж— йңҖеӨҮд»ҪгҖӮ

еӨҮд»Ҫж“ҚдҪң

еңЁдәҶи§ЈдәҶйңҖиҰҒеӨҮд»Ҫзҡ„ж–Үд»¶еҗҺпјҢе°ұеҸҜд»ҘиҝӣиЎҢеӨҮд»ҪдәҶпјҢеҸҜд»ҘжүӢеҠЁжүҫеҲ°иҝҷдәӣж–Үд»¶е№¶е°Ҷе…¶жӢ·иҙқеҮәжқҘпјҢжҲ–иҖ…дёҠдј еҲ°дә‘з«ҜпјҢдҪҶиҝҷж ·ж— з–‘жҳҜжҜ”иҫғйә»зғҰзҡ„гҖӮеӨҚеҲ¶иҝҷдёҖйҮҚеӨҚжҖ§ж“ҚдҪңеҸҜд»Ҙзј–еҶҷдёҖдёӘжү№еӨ„зҗҶи„ҡжң¬иҝӣиЎҢж“ҚдҪңпјҡ

echo d|xcopy "scaffolds" "../Back-up/MyBlog/scaffolds"  /E/I/Y

дҪҝз”Ёxcopyе‘Ҫд»ӨжқҘе®ҢжҲҗеӨҚеҲ¶ж“ҚдҪңпјҢж јејҸдёәпјҡxcopy <йңҖжӢ·иҙқзҡ„ж–Үд»¶и·Ҝеҫ„> <жӢ·иҙқеҗҺзҡ„ж–Үд»¶и·Ҝеҫ„>пјҢжңүж—¶зі»з»ҹеҸҜиғҪдјҡж— жі•еҲӨж–ӯи·Ҝеҫ„жҳҜж–Үд»¶еҗҚиҝҳжҳҜзӣ®еҪ•еҗҚпјҢеңЁжү§иЎҢж—¶дјҡи®©з”ЁжҲ·иҝӣиЎҢиҫ“е…ҘйҖүжӢ©пјҢиҫ“е…ҘdжҳҜзӣ®еҪ•еҗҚпјҢиҫ“е…ҘfжҳҜж–Үд»¶еҗҚгҖӮдёәдәҶйҳІжӯўзЁӢеәҸеӣ жӯӨдёӯж–ӯпјҢжҲ‘д»¬еҸҜд»ҘдҪҝз”Ёз®ЎйҒ“е‘Ҫд»ӨпјҡA | BпјҢиЎЁзӨәAе‘Ҫд»Өиҫ“еҮәзҡ„з»“жһңдҪңдёәBе‘Ҫд»Өзҡ„иҫ“е…ҘгҖӮ

иҝҷж ·жҲ‘д»¬дҫҝе°ҶйңҖиҰҒеӨҮд»Ҫзҡ„ж–Үд»¶жӢ·иҙқеҮәжқҘдәҶпјҢд№ӢеҗҺдҫҝеҸҜд»ҘдҝқеӯҳеңЁUзӣҳжҲ–иҖ…дёҠдј дә‘з«ҜпјҢиҝҷйҮҢжҲ‘йҖүжӢ©зӣҙжҺҘе°ҶеӨҮд»Ҫзҡ„ж–Үд»¶еӨ№жӢ–е…ҘOnedriveдёӯиҝӣиЎҢеҗҢжӯҘпјҢдёҖй”®ж“ҚдҪңпјҢз®ҖеҚ•еҝ«жҚ·гҖӮ

жҒўеӨҚ

еҰӮжһңз”өи„‘дёҠе·Із»Ҹе®үиЈ…дәҶhexoеҚҡе®ўпјҢз”ұдәҺжҹҗдәӣиҜҜж“ҚдҪңеҸӘжҳҜжғіжҒўеӨҚеҲ°д№ӢеүҚеӨҮд»Ҫзҡ„зҠ¶жҖҒзҡ„иҜқпјҢеҸӘйңҖиҰҒе°ҶеӨҮд»Ҫзҡ„йӮЈе…ӯдёӘж–Үд»¶е’Ңзӣ®еҪ•иҰҶзӣ–еҪ“еүҚеҚҡе®ўзӣёеә”зҡ„ж–Үд»¶е’Ңзӣ®еҪ•еҚіеҸҜгҖӮиҖҢеҰӮжһңжғіеңЁе…¶д»–з”өи„‘дёҠжҒўеӨҚеҚҡе®ўеҶ…е®№зҡ„иҜқпјҢйңҖиҰҒиҝӣиЎҢеҰӮдёӢж“ҚдҪңпјҡ

е®үиЈ…git bashе’Ңnode.js

е®үиЈ…еҗҺеҸҜд»ҘеңЁcmdдёӯиҫ“е…Ҙgit versionпјҢnode -vпјҢnpm -vжҹҘзңӢзүҲжң¬жқҘжЈҖжөӢжҳҜеҗҰжҲҗеҠҹе®үиЈ…гҖӮ

е®үиЈ…Hexo

еңЁcmdдёӯиҫ“е…Ҙnpm install -g hexo-cli

еҲқе§ӢеҢ–Hexo

еңЁжғіиҰҒдҝқеӯҳhexoзӣ®еҪ•зҡ„дҪҚзҪ®пјҢеҸій”®жү“ејҖgit bash hereпјҲд№ҹеҸҜд»Ҙcmdиҝӣе…ҘжӯӨзӣ®еҪ•пјүпјҢжү§иЎҢhexo init

жҒўеӨҚеӨҮд»Ҫж–Үд»¶

е°ҶеӨҮд»Ҫзҡ„е…ӯдёӘж–Үд»¶е’Ңзӣ®еҪ•иҰҶзӣ–еҲ°еҪ“еүҚеҚҡе®ўзӣ®еҪ•пјҢжҒўеӨҚж“ҚдҪңе®ҢжҲҗпјҢеҸҜд»Ҙhexo gпјҢhexo sиҝӣиЎҢжЈҖйӘҢгҖӮ
