MMIO е’Ң mmap ж–Үд»¶жҳ е°„йғҪж¶үеҸҠе°ҶеӨ–йғЁиө„жәҗпјҲйҷӨ CPU еҜ„еӯҳеҷЁе’ҢеҶ…еӯҳд»ҘеӨ–пјүжҳ е°„еҲ°иҝӣзЁӢзҡ„ең°еқҖз©әй—ҙдёӯпјҢдҪҶе®ғд»¬е№¶йқһзӯүеҗҢгҖӮжң¬ж–Үе°ҶеҜ№дәҢиҖ…иҝӣиЎҢеҲҶеҲ«д»Ӣз»ҚпјҢе№¶еҜ№жҜ”е®ғд»¬д№Ӣй—ҙзҡ„еҢәеҲ«е’ҢиҒ”зі»гҖӮ

MMIO

и®Ўз®—жңәзі»з»ҹдёӯзҡ„и®ҫеӨҮз§Қзұ»з№ҒеӨҡпјҢдёҚеҗҢи®ҫеӨҮе®ғд»¬зҡ„зү©зҗҶз»“жһ„гҖҒз”өж°”зү№жҖ§еҸҜиғҪе®Ңе…ЁдёҚеҗҢгҖӮеӣ жӯӨдёәдәҶеұҸи”Ҫе…¶дёӯзҡ„е·®ејӮпјҢи®© CPU жӣҙеҘҪзҡ„дёҺдёҚеҗҢзҡ„и®ҫеӨҮиҝӣиЎҢдәӨдә’пјҢеҗ„з§ҚдёҚеҗҢзҡ„и®ҫеӨҮйғҪиў«жҠҪиұЎжҲҗдәҶдёҖзі»еҲ—зҡ„еӨ–йғЁжҺҘеҸЈпјҢд№ҹе°ұжҳҜ и®ҫеӨҮеҜ„еӯҳеҷЁ гҖӮCPU дёҺи®ҫеӨҮдәӨдә’зҡ„ж–№ејҸпјҢе°ұжҳҜеҜ№иҝҷдәӣеҜ„еӯҳеҷЁиҝӣиЎҢиҜ»еҶҷгҖӮ

йӮЈд№Ҳй—®йўҳиҮӘз„¶еә”иҝҗиҖҢз”ҹпјҢд»Ҙд»Җд№Ҳж ·зҡ„ж–№ејҸпјҢжҲ–иҖ…жӣҙе…·дҪ“ең°иҜҙпјҢжү§иЎҢд»Җд№Ҳж ·зҡ„жҢҮд»ӨеҜ№и®ҫеӨҮеҜ„еӯҳеҷЁиҝӣиЎҢиҜ»еҶҷпјҹйҖҡеёёжқҘиҜҙжңүдёӨз§ҚпјҡдёҖз§ҚжҳҜ CPU йҮҮз”Ёзү№ж®Ҡзҡ„ I/O жҢҮд»ӨпјҲеҰӮ x86 жһ¶жһ„дёӢзҡ„ in/out жҢҮд»ӨпјүпјҢжҢҮд»ӨдёӯеҜ№еә”зҡ„и®ҫеӨҮеҜ„еӯҳеҷЁең°еқҖдҪҚдәҺдёҖзүҮдё“й—Ёзҡ„ I/O ең°еқҖз©әй—ҙдёӯпјҢиҝҷз§Қж–№ејҸжҲҗдёәз«ҜеҸЈжҳ е°„ I/OпјҲPort-Mapped I/O, PMIOпјүпјӣиҖҢеҸҰдёҖз§ҚеҲҷжҳҜе°Ҷи®ҫеӨҮеҜ„еӯҳеҷЁзҡ„ең°еқҖжҳ е°„еҲ°иҝӣзЁӢзҡ„иҷҡжӢҹең°еқҖз©әй—ҙдёӯпјҢиҝӣиҖҢзӣҙжҺҘдҪҝз”Ёжҷ®йҖҡзҡ„еҶ…еӯҳиҜ»еҶҷжҢҮд»ӨпјҲеҰӮ x86 жһ¶жһ„дёӢзҡ„ movпјүеҜ№е…¶иҝӣиЎҢиҜ»еҶҷпјҢиҝҷз§Қж–№ејҸз§°дёәеҶ…еӯҳжҳ е°„ I/OпјҲMemory-Mapped I/OпјүпјҢд№ҹе°ұжҳҜжң¬ж–ҮиҰҒд»Ӣз»Қзҡ„ MMIOгҖӮ

mmap ж–Үд»¶жҳ е°„

еұһдәҺ MMIO зҡ„дёҖз§ҚпјҹгҖҠUNIX зҺҜеўғй«ҳзә§зј–зЁӢ p422гҖӢ

mmap жҳҜдёҖдёӘзұ» UNIX зі»з»ҹдёӯеёёи§Ғзҡ„зі»з»ҹи°ғз”ЁжҺҘеҸЈпјҢз”ЁдәҺеҗ‘иҝӣзЁӢзҡ„ең°еқҖз©әй—ҙдёӯж–°еўһдёҖж®өиҷҡжӢҹеҶ…еӯҳж®өпјҢе…¶еЈ°жҳҺеҰӮдёӢпјҡ

еҰӮжһңжҢҮе®ҡдәҶ fd еҸӮж•°пјҲfd дёҚдёә -1пјүпјҢйӮЈд№Ҳ fd жүҖжҢҮеҗ‘зҡ„ж–Үд»¶е°Ҷдјҡд»Һ off еҒҸз§»йҮҸзҡ„дҪҚзҪ®ејҖе§Ӣжҳ е°„дёҖж®өй•ҝеәҰдёә len зҡ„ж•°жҚ®еҶ…е®№еҲ°ең°еқҖз©әй—ҙдёӯпјҢеҜ№ж–Үд»¶зҡ„иҜ»еҶҷеҸҜд»ҘзӣҙжҺҘиҪ¬жҚўдёәеҜ№иў«жҳ е°„еҶ…еӯҳең°еқҖзҡ„иҜ»еҶҷгҖӮеҪ“з„¶иҝҷдёӘиҝҮзЁӢйҖҡеёёеёҰжңү lazy зҡ„зӯ–з•ҘпјҢеҚіеҸӘжңүе®һйҷ…иҜ»еҸ–ж—¶жүҚиҝӣиЎҢеҠ иҪҪгҖӮзӨәдҫӢд»Јз ҒеҰӮдёӢпјҡ

дәҢиҖ…зҡ„еҢәеҲ«е’ҢиҒ”зі»

еҸҜд»ҘзңӢеҲ°пјҢдәҢиҖ…еңЁдҪҝз”ЁдёҠзҡ„еҢәеҲ«иҝҳжҳҜеҫҲжҳҫи‘—зҡ„гҖӮmmap ж“ҚдҪңзҡ„жҳҜжҢҒд№…еҢ–зҡ„еӯҳеӮЁпјҲж–Үд»¶пјүпјҢзӣ®зҡ„жҳҜж–№дҫҝеҜ№ж–Үд»¶ж•°жҚ®зҡ„иҜ»еҶҷпјӣMMIO ж“ҚдҪңзҡ„жҳҜзЎ¬д»¶иЎҢдёәпјҢзӣ®зҡ„жҳҜжҺ§еҲ¶и®ҫеӨҮзҡ„зҠ¶жҖҒгҖӮ

дҪҶжҳҜжҲ‘еңЁйҳ…иҜ»гҖҠUNIX зҺҜеўғй«ҳзә§зј–зЁӢпјҡз¬¬ 3 зүҲгҖӢпјҲдёӢйқўз®Җз§° APUEпјүз¬¬ 14.8 иҠӮвҖ”вҖ”еӯҳеӮЁжҳ е°„ I/O ж—¶пјҢе…¶дёӯд»Ӣз»Қзҡ„еҶ…е®№жӯЈжҳҜ mmap жҺҘеҸЈгҖӮе®һйҷ…дёҠпјҢAPUE дёӯжүҖжҸҸиҝ°зҡ„ MMIOпјҢжҢҮзҡ„еә”иҜҘжҳҜ е№ҝд№үдёҠ зҡ„ MMIO вҖ”вҖ” еҚійҖҡиҝҮеҶ…еӯҳең°еқҖе®һзҺ° I/O ж“ҚдҪңгҖӮе°ҶеӯҳеӮЁеңЁзЈҒзӣҳдёӯзҡ„ж–Үд»¶жҳ е°„еҲ°ең°еқҖз©әй—ҙдёӯпјҢе…¶е®һд№ҹжҳҜеңЁе°ҶзЈҒзӣҳзҡ„жҹҗдәӣжүҮеҢәжҳ е°„еҲ°ең°еқҖз©әй—ҙдёӯпјҢйӮЈд№ҲеҜ№иҝҷдәӣең°еқҖзҡ„еҶҷе…ҘпјҢжңҖз»Ҳд№ҹдјҡиҪ¬жҚўдёәеҜ№ж–Үд»¶зҡ„еҶҷе…ҘпјҲеҰӮжһңдҪҝз”Ё MAP_SHARED жҳ е°„зұ»еһӢпјүпјҢеҜ№ж–Үд»¶зҡ„еҶҷе…ҘпјҢйңҖиҰҒйҖҡиҝҮи°ғз”ЁзЈҒзӣҳй©ұеҠЁзЁӢеәҸжқҘеҜ№жӣҙж”№зЈҒзӣҳдёҠжҢҮе®ҡзҡ„жҜ”зү№дҪҚпјҢеҖҹжӯӨе®һзҺ°жҢҒд№…еҢ–пјҢиҝҷе…¶е®һд№ҹжҳҜеңЁжӣҙж”№и®ҫеӨҮзҡ„зҠ¶жҖҒгҖӮд»ҺиҝҷдёӘи§’еәҰжқҘзңӢпјҢmmap ж–Үд»¶жҳ е°„д№ҹжҳҜ MMIO зҡ„дёҖз§ҚгҖӮ
