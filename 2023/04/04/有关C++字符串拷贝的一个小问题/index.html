жңҖиҝ‘йқўиҜ•иў«й—®еҲ°дәҶдёҖдёӘ C++ дёӯзҡ„е°Ҹй—®йўҳпјҢе°ұжҳҜеҰӮжһңжңүеӯ—з¬ҰдёІ s1 е’Ң s2пјҢе°Ҷ s1 иөӢеҖјз»ҷ s2 еҗҺпјҢе®ғд»¬зҡ„еҶ…еӯҳеҲҶеёғжҳҜд»Җд№Ҳж ·зҡ„гҖӮеҪ“ж—¶ж„ҹи§үеҸҜиғҪжҳҜе…ұдә«зҡ„пјҢдҪҶд№ҹдёҚеӨӘзЎ®е®ҡпјҢеӣһжқҘеҗҺжҹҘйҳ…иө„ж–ҷеҸ‘зҺ°з»“жһңе№¶дёҚжҳҜйӮЈд№Ҳз®ҖеҚ•гҖӮ

жҹҘйҳ…иө„ж–ҷ

йҰ–е…ҲжҳҜиҜўй—® chatGPT еҫ—еҲ°зҡ„зӯ”жЎҲпјҢеҰӮдёӢпјҡ

еҰӮжһңдёӨдёӘ string еҜ№иұЎеӯҳеӮЁзӣёеҗҢзҡ„еӯ—з¬ҰдёІпјҢе®ғд»¬еҸҜиғҪдјҡе…ұдә«еҗҢдёҖдёӘеҶ…еӯҳеқ—пјҢд№ҹеҸҜиғҪдјҡеҲҶеҲ«еҲҶй…ҚиҮӘе·ұзҡ„еҶ…еӯҳеқ—гҖӮеҪ“дёҖдёӘ string еҜ№иұЎиў«еҲӣе»әж—¶пјҢе®ғдјҡеҲҶй…ҚдёҖдёӘж–°зҡ„еҶ…еӯҳеқ—пјҢе№¶е°Ҷеӯ—з¬ҰдёІеӨҚеҲ¶еҲ°иҜҘеҶ…еӯҳеқ—дёӯгҖӮеҪ“з¬¬дәҢдёӘ string еҜ№иұЎиў«еҲӣе»әж—¶пјҢе®ғеҸҜиғҪдјҡдҪҝз”ЁдёҺз¬¬дёҖдёӘ string еҜ№иұЎзӣёеҗҢзҡ„еҶ…еӯҳеқ—пјҢд№ҹеҸҜиғҪдјҡеҲҶй…ҚдёҖдёӘж–°зҡ„еҶ…еӯҳеқ—е№¶е°Ҷеӯ—з¬ҰдёІеӨҚеҲ¶еҲ°иҜҘеҶ…еӯҳеқ—дёӯгҖӮ

д»Јз ҒжөӢиҜ•

зңӢжқҘеҸҲжҳҜдёҖдёӘжІЎжңүж ҮеҮҶи§ЈеҶіж–№жЎҲзҡ„й—®йўҳпјҢдәҺжҳҜжҲ‘зј–еҶҷдәҶдёҖдёӘз®ҖеҚ•зҡ„зЁӢеәҸз”ЁжқҘеңЁдёҚеҗҢзҡ„зҺҜеўғдёӢжөӢиҜ•гҖӮ

void* operator new(size_t count) {
	cout << "Allocate " << count << " Bytes" << endl;
	return malloc(count);
}

int main()
{
	string s1 = "Hello World";
	string s2 = s1;

	cout << (void*)(s1.c_str()) << endl;
	cout << (void*)(s2.c_str()) << endl;

	s2[0] = 'h';
	cout << (void*)(s1.c_str()) << endl;
	cout << (void*)(s2.c_str()) << endl;
}

еңЁдёҠйқўзҡ„зЁӢеәҸдёӯпјҢжҲ‘йҮҚиҪҪдәҶ operator new еҮҪж•°жқҘиҝҪиёӘеҶ…еӯҳзҡ„еҲҶй…ҚпјҢз„¶еҗҺе…ҲеҲқе§ӢеҢ–еӯ—з¬ҰдёІ s1пјҢз„¶еҗҺеҶҚе°Ҷ s1 дҪңдёә s2 зҡ„еҲқе§ӢеҖјиҝӣиЎҢиөӢеҖјпјҢиҫ“еҮә s1 е’Ң s2 еӯ—з¬ҰдёІзҡ„ең°еқҖжҳҜеҗҰжҳҜеҗҢдёҖеқ—пјҢеҚіеҶ…еӯҳжҳҜеҗҰе…ұдә«пјҢз„¶еҗҺжҲ‘еҶҚдҝ®ж”№ s2 зҡ„йҰ–еӯ—жҜҚпјҲд№ӢжүҖд»Ҙиҝҷд№ҲеҒҡжҳҜеӣ дёәдҝ®ж”№ s2 еҗҺпјҢз”ұдәҺ s1 дёҺ s2 иҰҒдҝқжҢҒзӢ¬з«ӢжҖ§пјҢеӣ жӯӨеҰӮжһңд№ӢеүҚеҶ…еӯҳжҳҜе…ұз”Ёзҡ„пјҢйӮЈд№ҲеңЁдҝ®ж”№д№ӢеүҚеҝ…е®ҡйңҖиҰҒдёә s2 йҮҚж–°еҲҶй…ҚеҶ…еӯҳпјҢеҗҰеҲҷж”№еҸҳ s2 еҗҺ s1 д№ҹе°Ҷж”№еҸҳпјүпјҢеҶҚиҫ“еҮәжҹҘзңӢ s1 е’Ң s2 еӯ—з¬ҰдёІзҡ„ең°еқҖжҳҜеҗҰзӣёеҗҢгҖӮ

д»ҘдёӢжҳҜеңЁдёҚеҗҢзј–иҜ‘еҷЁдёӢзЁӢеәҸиҝҗиЎҢзҡ„з»“жһңпјҡ

g++

MSVC

еңЁ g++ дёӢжңӘеҜ№еӯ—з¬ҰдёІиҝӣиЎҢжӣҙж”№ж—¶пјҢs1 е’Ң s2 е…ұз”ЁдёҖеқ—еҶ…еӯҳз©әй—ҙпјҢиҖҢеңЁдҝ®ж”№еүҚжүҚдёә s2 еҲҶй…ҚеҶ…еӯҳз©әй—ҙгҖӮиҖҢеңЁ MSVC дёӢ s1 е’Ң s2 е§Ӣз»ҲжІЎжңүе…ұз”ЁеҶ…еӯҳз©әй—ҙгҖӮеҸҜи§ҒдёҚеҗҢзј–иҜ‘еҷЁеҜ№жӯӨжғ…еҶөзҡ„еӨ„зҗҶдёҚе°ҪзӣёеҗҢгҖӮ

еңЁ g++ дёӯпјҢеҜ№еӯ—з¬ҰдёІжӢ·иҙқдҪҝз”ЁдәҶ COWпјҲCopy-On-WriteпјүжҠҖжңҜпјҢеҚіеҪ“дёҖдёӘ string еҜ№иұЎиў«еӨҚеҲ¶ж—¶пјҢдёҚдјҡз«ӢеҚіеӨҚеҲ¶еҶ…еӯҳеқ—пјҢиҖҢжҳҜе…ұдә«еҗҢдёҖдёӘеҶ…еӯҳеқ—пјҢзӣҙеҲ°е…¶дёӯдёҖдёӘ string еҜ№иұЎиҰҒдҝ®ж”№еӯ—з¬ҰдёІеҶ…е®№ж—¶пјҢжүҚдјҡеӨҚеҲ¶еҶ…еӯҳеқ—е№¶еҲҶзҰ»еҮәдёҖдёӘж–°зҡ„еҶ…еӯҳеқ—гҖӮиҝҷж ·еҸҜд»ҘйҒҝе…Қйў‘з№Ғзҡ„еҶ…еӯҳеҲҶй…Қе’ҢеӨҚеҲ¶пјҢжҸҗй«ҳзЁӢеәҸзҡ„жҖ§иғҪгҖӮеӣ жӯӨпјҢеҰӮжһңдёӨдёӘ string еҜ№иұЎеӯҳеӮЁзӣёеҗҢзҡ„еӯ—з¬ҰдёІпјҢе®ғд»¬еҸҜиғҪдјҡе…ұдә«еҗҢдёҖдёӘеҶ…еӯҳеқ—гҖӮ

иҖҢеңЁ MSVC дёӯпјҢеҲҷдёҚйҮҮз”Ё COW жҠҖжңҜпјҢиҖҢжҳҜжҜҸдёӘ string еҜ№иұЎйғҪеҲҶй…ҚиҮӘе·ұзҡ„еҶ…еӯҳеқ—жқҘеӯҳеӮЁеӯ—з¬ҰдёІгҖӮеӣ жӯӨпјҢеҰӮжһңдёӨдёӘ string еҜ№иұЎеӯҳеӮЁзӣёеҗҢзҡ„еӯ—з¬ҰдёІпјҢе®ғд»¬дјҡеҲҶеҲ«еҲҶй…ҚиҮӘе·ұзҡ„еҶ…еӯҳеқ—гҖӮ

COW зҡ„зјәзӮ№

MSVC д№ӢжүҖд»ҘдёҚйҮҮз”Ё COW жҠҖжңҜпјҢеҸҜиғҪжҳҜеӣ дёә COW жҠҖжңҜдјҡеёҰжқҘдёҖдәӣйўқеӨ–зҡ„ејҖй”Җе’ҢеӨҚжқӮеәҰгҖӮ

йҰ–е…ҲпјҢCOW жҠҖжңҜйңҖиҰҒйўқеӨ–зҡ„д»Јз ҒжқҘе®һзҺ°еҶ…еӯҳе…ұдә«е’ҢеӨҚеҲ¶пјҢиҝҷдјҡеёҰжқҘдёҖе®ҡзҡ„жҖ§иғҪејҖй”Җе’ҢеҶ…еӯҳеҚ з”ЁгҖӮеңЁжҹҗдәӣжғ…еҶөдёӢпјҢиҝҷз§ҚејҖй”ҖеҸҜиғҪдјҡи¶…иҝҮдҪҝз”ЁеҠЁжҖҒеҶ…еӯҳеҲҶй…Қзҡ„ејҖй”ҖгҖӮ

е…¶ж¬ЎпјҢCOW жҠҖжңҜдјҡеўһеҠ д»Јз Ғзҡ„еӨҚжқӮеәҰе’ҢйҡҫеәҰгҖӮз”ұдәҺйңҖиҰҒеӨ„зҗҶеҶ…еӯҳе…ұдә«е’ҢеӨҚеҲ¶пјҢд»Јз Ғзҡ„е®һзҺ°дјҡеҸҳеҫ—жӣҙеҠ еӨҚжқӮе’Ңйҡҫд»Ҙз»ҙжҠӨгҖӮжӯӨеӨ–пјҢCOW жҠҖжңҜиҝҳеҸҜиғҪдјҡеёҰжқҘдёҖдәӣжҪңеңЁзҡ„йЈҺйҷ©пјҢеҰӮзәҝзЁӢе®үе…ЁжҖ§й—®йўҳзӯүгҖӮ

еӣ жӯӨпјҢдёҚеҗҢзҡ„зј–иҜ‘еҷЁе®һзҺ°еҸҜиғҪдјҡж №жҚ®е…·дҪ“жғ…еҶөжқҘйҖүжӢ©жҳҜеҗҰйҮҮз”Ё COW жҠҖжңҜгҖӮеңЁдёҖдәӣжғ…еҶөдёӢпјҢCOW жҠҖжңҜеҸҜд»ҘжҸҗй«ҳжҖ§иғҪе’Ңж•ҲзҺҮпјҢдҪҶеңЁеҸҰдёҖдәӣжғ…еҶөдёӢпјҢе®ғеҸҜиғҪдјҡеёҰжқҘйўқеӨ–зҡ„ејҖй”Җе’ҢеӨҚжқӮеәҰгҖӮ
